# Halal Crypto Portfolio Website

This is a React + Vite portfolio website for the **Halal Crypto Intelligence System** project.

## Run locally

```bash
npm install
npm run dev
```

## Build for production

```bash
npm run build
```

## Image setup

Place your screenshots inside:

```text
public/assets/images/
```

Use these exact filenames:

- dashboard.png
- filters.png
- coin_breakdown.png
- feature_importance.png
